## 方法二：本地运行项目
本项目使用 HBuilderX 打开。

**打开步骤**
1. 下载 HBuilderX
2. 文件 → 打开目录
3. 选择本项目文件夹

**运行步骤**
1. 点击运行
2. 运行到浏览器
3. Edge 或 Chrome

注意：不要直接双击 `index.html` 打开，否则页面可能无法正常跳转。

## 方法三：运行到手机真机

如果要测试拍照识诗、横屏效果、手机端页面交互，建议运行到手机真机。

**安卓 / 鸿蒙手机调试步骤**

1. 手机打开开发者模式
2. 打开 USB 调试
3. 数据线连接电脑
4. USB 连接方式选择“传输文件 / MTP”
5. 手机弹出“是否允许 USB 调试”时点击允许
6. HBuilderX 点击：运行 → 运行到手机或模拟器
7. 选择已连接的手机设备

注意：鸿蒙手机如果找不到设备，可以检查“开发人员选项”里的 USB 调试、仅充电模式下允许 ADB 调试、USB 安装权限是否打开。

# 二、前端和后端联调
### 正常联调
前端接口统一写在：`utils/api.js`

当前本地接口地址：
```js
const BASE_URL = 'http://127.0.0.1:8000'
```

**联调顺序**
1. VSCode 启动后端 FastAPI
2. 浏览器访问 `http://127.0.0.1:8000/ping`
3. HBuilderX 运行前端到 Edge 或 Chrome
4. 在前端页面测试功能

如果 `/ping` 返回 `pong`，说明后端启动成功。

---

### 手机真机联调

如果前端运行在手机上，`127.0.0.1` 指的是手机自己，不是电脑。

所以手机真机联调时，需要把 `utils/api.js` 里的地址改成电脑局域网 IP，例如：

```js
const BASE_URL = 'http://192.168.1.23:8000'
```

后端启动时也要允许局域网访问：

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```
mine:http://192.168.43.235:8000
电脑和手机需要连接同一个 Wi-Fi。

# 三、前端接口对接情况

| 前端功能 | 后端接口 | 对接状态 | 说明 |
| --- | --- | --- | --- |
| 后端连接检测 | GET /ping | 已接入 | 检查后端是否启动 |
| 搜索古诗 | GET /poems/search | 已接入 | 支持诗名、作者、内容、标签搜索 |
| 古诗详情 | GET /poems/{poem_id} | 已接入 | 根据诗词 ID 获取详情 |
| 添加学习记录 | POST /record | 已接入 | 学完后记录学习时长 |
| 家长端学习统计 | GET /record/summary | 已接入 | 前端会过滤 0 秒记录，展示每首诗最新学习时长和累计学习时长 |
| 推荐古诗 | GET /recommend | 已接入 | 根据学习记录推荐 |
| AI 诗人对话 | POST /chat | 已接入，依赖后端配置 | 需要后端配置 vivo API，否则会提示接口不可用 |
| 拍照识诗 | POST /ocr | 已接入 | 手机端通过系统相机或相册上传图片，前端传 base64 给后端识别 |
| AI 配图生成 | POST /generate/image | 已接入 | 进入学习页前会预热生成；播放页显示 loading，生成完成后自动播放分镜图片 |
| 语音朗读 | POST /tts | 已接入 | 学习页图片开始播放后自动朗读诗词，中途退出会停止语音 |
| 诗人形象生成 | POST /generate/peot_avatar | 已接入 | 对话页、学习完成页优先使用后端诗人头像，失败时用本地默认头像 |

## 说明
1. 不要直接修改 `docs` 里的代码，源码在 `pages`、`utils`、`static`。
2. 修改源码后需要重新打包 H5 并覆盖 `docs` 才能更新在线预览。
3. `unpackage` 是临时目录，不需要上传。
4. 接口失败优先检查：后端是否启动、地址是否正确、AI 配置是否正常。
5. 本地调试地址：`http://127.0.0.1:8000`。
6. 打包安卓 APK 时，需要改成电脑局域网 IP。
7. 页面样式异常请在 HBuilderX 重新运行，不要直接打开 index.html。
8.  `/generate/image` 首次生成较慢，前端请求超时时间必须设置为 120 秒以上。
9. 诗人头像静态资源路径为 `/static/images/poets/诗人名.jpg`，例如 `/static/images/poets/李白.jpg`。
